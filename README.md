# Wardon
## A Ideia
O Wardon surgiu como uma ideia para um projeto da matéria Levantamento de Requisitos, do curso de Desenvolvimento de Sistemas do SENAI. Identificamos um problema na escola SESI, onde os alunos, mesmo com regras claras proibindo, continuavam utilizando seus celulares em ambiente escolar. A partir deste problema, desenvolvemos o Wardon para suprir essa necessidade de controle sobre os celulares; implantaremos armários inteligentes com carregadores por indução, para que, ao chegar na escola, o aluno escaneie seu rosto no sistema, abre o próprio armário e coloque o seu celular.

## Como Funcionará
O aluno, ao colocar o celular no armário e no carregador por indução, o sistema identifica que o mesmo colocou seu aparelho no armário. Se algum aluno tentar violar o sistema ou os armários, a diretoria da unidade receberá uma notificação imediatamente.

## Impactos a longo prazo
À longo prazo, o uso de celular será reduzido ao nulo, pois, guardando os aparelhos e denunciando os alunos que violam os armários, poderemos controlar o uso de celulares totalmente.
